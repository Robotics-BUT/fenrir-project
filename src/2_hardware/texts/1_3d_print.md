# 3D Printed Components

This chapter presents all parts to be 3D printed. 

Recommendations
 - Printer: Prusa MK4
 - Slicer: Prusa Slicer
 - Filament: Prusament PLA or PETG
 - Infill: 30%
 - Layer height: 0.2mm
 - Infill pattern: gyroid
